# Instrumentation

The simulator records structured trace events during execution.

## Recorded Events

- memory accesses
- cache hits
- cache misses
- line loads
- evictions

## Metrics

Collected metrics include:

- hit rate
- miss rate
- eviction count
- per-address access frequency
- per-cache-line access frequency
- simulated cycle cost

## Analysis

Instrumentation data powers:

- timeline visualization
- heatmaps
- cache pressure analysis
- benchmark comparison
